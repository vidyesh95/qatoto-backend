# Qatoto Backend

Qatoto is a platform for product research, development and support.

## Getting Started

### Prerequisites

* Node.js >= 24.13.1
* pnpm >= 10.29.3

### Installation

```bash
pnpm install
```

### Running the Server

```bash
pnpm run dev
```

### Running Tests

```bash
pnpm test
```

### Formatting

```bash
pnpm run fmt
```

### Linting

```bash
pnpm run lint
```
